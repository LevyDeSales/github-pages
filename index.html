Repositório de Arquivos HTML

Este é um simples repositório para hospedar arquivos HTML no GitHub Pages

Como adicionar novos arquivos

Para adicionar novos arquivos HTML a este repositório:

  1. Faça um clone do repositório
  2. Adicione seus arquivos HTML na pasta htmls/
  3. Faça commit e push das alterações
  4. O GitHub Actions atualizará automaticamente esta página e listará os novos arquivos

Importante: O GitHub Pages é apenas para hospedagem estática, não é possível fazer upload de arquivos diretamente pelo navegador.
